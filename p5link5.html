Project Overview

This project involves creating a web application that allows users to convert their unused coupons into cash. The platform offers an easy and intuitive interface where users can input details of their coupons and receive an estimated cash value, encouraging the use of otherwise wasted resources.

Key Features

  • Easy-to-use interface for entering coupon details and viewing estimated cash values.
  • Support for various coupon types, including retail, dining, and online offers.
  • Secure processing of user data with encrypted communications.

Technologies Used

  • React: For building the interactive user interface.
  • Node.js & Express: For server-side processing and backend logic.
  • MongoDB: Database for storing user data and coupon information.
